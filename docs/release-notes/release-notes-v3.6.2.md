# v3.6.2 - 修复图层控件点击常驻

本版本修复 GeoTIFF 地图图层控件在鼠标移动时自动收起的问题，统一普通图源与 Wayback 历史影像的图层面板交互。

## 修复内容

- 图层控件改为点击展开并常驻，点击面板外或按 Esc 关闭。
- 修复 GeoTIFF 模式下鼠标移动导致地图控件被重建、面板看起来自动收起的问题。
- 普通图源控件与 Wayback 控件使用一致的点击式交互，避免 Leaflet 原生 hover 折叠行为影响选择图层。
- 保持图层面板内的底图与天地图标注切换行为不变。

## 验证

- 通过前端生产构建。
- 已验证 GeoTIFF 初始状态下点击图层控件后，鼠标移动到面板内部不会关闭面板。
