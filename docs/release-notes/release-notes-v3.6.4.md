# v3.6.4 — 经纬度网格

本版新增地图经纬度网格显示，方便在下载前查看区域所处的经纬度范围与网格位置。

## 新增

- **经纬度网格开关**
  地图右上角新增「经纬网」开关，可随时显示或隐藏经纬度网格。

- **自定义网格间隔**
  支持选择 `0.1°`、`0.25°`、`0.5°`、`1°`、`2°`、`5°`、`10°`、`15°`、`30°` 等间隔。

- **经纬度标签**
  经纬线带有轻量坐标标签，例如 `116°E`、`39°N`。标签靠近视野边缘显示，减少对地图主体内容的遮挡。

## 体验优化

- 网格线使用「浅色描边 + 深色虚线」双层绘制，在影像底图和浅色矢量底图上都更容易辨认。
- 视野过大时会自动放大实际显示间隔，避免密集网格影响性能和可读性，并在控制面板中提示当前实际间隔。
- 经纬网开关和间隔会自动记住，下次打开应用时保持上次设置。

## 验证

- 前端 `npm run build` 通过。
